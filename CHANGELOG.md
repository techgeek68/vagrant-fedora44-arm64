## [1.0.0] - Initial Release

### Added
- Initial publication of `techgeek68/fedora44-arm64` on Vagrant Cloud.
- Fedora Server 44 (aarch64) base image.
- VirtualBox Guest Additions (ARM64 build) preinstalled.
- VirtIO SCSI storage controller configuration for reliable disk and DVD handling on ARM64 guests.
- NAT networking with automatic SSH port forwarding.
- Documentation repository including installation, quick start, usage, configuration, troubleshooting, and FAQ guides.
